# Carnival 3D Engine

A 3D Graphics Engine built from scratch using C++ and OpenGL 4.1. This project was developed as part of my academic journey in Computer Graphics, focusing on real-time rendering techniques and engine architecture.

## Technical Features
- **Custom Shader Pipeline:** Implemented Vertex and Fragment shaders using GLSL.
- **Advanced Lighting:** Real-time lighting system including Shadow Mapping for depth and realism.
- **Model Loading:** Integrated model loading for complex 3D geometries (GLTF).
- **Environment Mapping:** Support for Skyboxes and Cubemaps for immersive environments.
- **Input Handling:** Smooth camera movement and user interaction system.

## Built With
- **Language:** C++17
- **Graphics API:** OpenGL 4.1
- **Tools:** Visual Studio 2022
- **Libraries:** GLFW, GLM, Glad

## Project Structure
- `Carnival-ProjectGP/` - Core engine source code: classes (.cpp, .h), shaders (.glsl), and internal resources.
- `Carnival-ProjectGP.slnx` - Visual Studio solution file to open and build the project.
- `Carnival_GP_Doc.pdf` - Comprehensive technical documentation and project report.
- `README.md` - Project overview and documentation.

## How to Run
1. Clone the repository.
2. Open `Carnival-ProjectGP.slnx` in Visual Studio 2022.
3. Ensure all dependencies are correctly linked in the project properties.
4. Build and Run in **Debug** or **Release** (x64).

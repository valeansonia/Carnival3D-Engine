# Carnival 3D Engine 🎡

A 3D Graphics Engine built from scratch using **C++** and **OpenGL 4.1**. This project was developed as part of my academic journey in Computer Graphics, focusing on real-time rendering techniques and engine architecture.

## 🚀 Technical Features
* **Custom Shader Pipeline:** Implemented Vertex and Fragment shaders using **GLSL**.
* **Advanced Lighting:** Real-time lighting system including **Shadow Mapping** for depth and realism.
* **Model Loading:** Integrated model loading for complex 3D geometries (using GLTF/Assimp).
* **Environment Mapping:** Support for **Skyboxes** and **Cubemaps** to create immersive environments.
* **Input Handling:** Smooth camera movement and user interaction system.

## 🛠️ Built With
* **Language:** C++17
* **Graphics API:** OpenGL 4.1
* **Tools:** Visual Studio 2022
* **Libraries:** GLFW (Window management), GLM (Mathematics), Glad (OpenGL loader).

## 📂 Project Structure
* `Carnival-ProjectGP/` - Core engine classes, logic, and GLSL shaders.
* `assets/` - 3D models and scene configurations.
* `cubemap/` - Skybox textures and environment maps.
* `imgdoc/` - Project documentation and screenshots.

## ⚙️ How to Run
1. Clone the repository.
2. Open `Carnival-ProjectGP.slnx` in Visual Studio 2022.
3. Ensure all dependencies are linked correctly.
4. Build and Run in **Debug** or **Release** (x64).
